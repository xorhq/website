Centralized, Automated, Integrated

Forexor, your one place for managing foreign exchange displays.

Unique Features

A centralized solution for managing rate exchange LED displays automatically without agents intervention.

Web Portal

A central solution web portal allowing easy management of displays templaes and sending data for specific diplays.

Platform agnostic

Our server and local agent are platform agnostic*, meaning you can run both on Windows, Mac and linux.

Cloud option

You get the latest edge updates by using the cloud version, no servers or IT team required for setup and maintenance.

API & Integration

Achieve full automation with a our easy to use REST based API, integrate with your existing systems for easy workflow.

How it works

We provide an agnostic solution with small agent running on devices connected to the Exchange display, a centralized web portal for changing rates data, and an easy API for seamless integration with existing systems.
